# Women's Day Celebration App

A beautiful, interactive Women's Day celebration app built with React, Vite, and Tailwind CSS.

## Features

✨ **Elegant Design**: Pink and gold theme with floating flowers animation
🎵 **Background Music**: Customizable background music with volume controls
🎊 **Interactive Elements**: Confetti celebration on button click
🌹 **Responsive Layout**: Works beautifully on all devices
🎨 **Custom Typography**: Beautiful Google Fonts integration

## Tech Stack

- **React 18** - Modern React with hooks
- **Vite** - Fast build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Smooth animations
- **Canvas Confetti** - Celebration effects
- **Lucide React** - Beautiful icons

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser

## Build for Production

```bash
npm run build
```

## Customization

- **Background Image**: Replace `public/womens_day_poster.jpg`
- **Background Music**: Replace `public/background-music.mp3`
- **Colors**: Modify Tailwind config in `tailwind.config.js`
- **Fonts**: Update font imports in `src/index.css`
